# 项目背景

HTMLParser 是一个 C++ HTML 解析库项目，定位是轻量、快速、可用于工程环境的 HTML 内容解析与处理。它并不追求过度庞杂的抽象，而是更重视在 C++ 里提供一套直接、实用的解析能力。

## 设计目标

- 提供高性能的 HTML 解析能力
- 保持轻量与灵活，方便集成到现有工程
- 支持网页抓取、文档转换等实际场景

## 核心功能

- HTML 内容解析与处理
- 适合 C++ 场景集成
- 面向网页抓取与文档转换等使用方向
- 以 CMake 组织构建
- 规划继续补充元素支持、错误处理与单元测试

## 当前状态与未来计划

当前项目已经形成基础解析库定位，并明确了后续路线，包括补充更多 HTML 元素与属性支持、改进错误处理，以及增加解析函数的单元测试。整体上属于一个很适合继续打磨内核能力的基础库项目。
