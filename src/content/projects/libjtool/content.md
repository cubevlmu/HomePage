# 项目背景

libJTool 是一个面向 JNI 开发的轻量级 C/C++ 工具库，目标是把 JNI 常见的样板代码、对象访问与回调处理整理成更清晰、更安全的接口。它服务的不是“做更多功能”，而是“让 JNI 开发本身没那么痛苦”。

## 设计目标

- 降低 JNI 开发的样板代码负担
- 提供更清晰的 Java 与 C/C++ 交互方式
- 以模块化设计支持不同 JNI 使用场景

## 核心功能

- 简化 Java 方法调用与字段访问
- 提供 C++ 与 Java 间的回调支持
- 使用面向对象风格封装 Java 对象
- 集成轻量日志模块辅助调试
- 通过 JniHelper、JniFields、JniCallbacks 等组件保持模块化

## 当前状态与未来计划

当前 libJTool 已经明确了“让 JNI 更易用”的产品方向，功能上覆盖了对象封装、回调与日志等典型痛点。后续可以继续围绕 API 稳定性、示例、文档与更多 JNI 场景适配推进，让它成为更成熟的基础开发库。
