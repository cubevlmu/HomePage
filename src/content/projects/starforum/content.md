# 项目背景

StarForum 是一个面向 Flarum 社区的跨平台论坛客户端，目标不是简单包一层网页，而是提供在移动端与桌面端都更顺手、更现代、也更贴近原生应用体验的论坛访问方式。项目强调性能、桌面适配、本地缓存与 Material 3 风格的一体化体验。

## 设计目标

- 为 Flarum 社区提供真正可用的跨平台客户端
- 在桌面与移动端之间保持一致但不生硬的交互体验
- 兼顾性能、缓存、页面切换与阅读体验
- 用更完整的主题、个性化和多语言能力提升产品完成度

## 核心功能

- 支持 Android、iOS、Windows、macOS 与 Linux
- 基于 Material 3 的界面与主题个性化设置
- 桌面端分栏浏览与详情页状态保留
- 首次加载骨架屏与更安静的页面过渡
- 论坛帖子详情渲染与图片加载优化
- 通知中心、搜索、用户页、分类浏览等完整论坛能力

## 当前状态与未来计划

目前 StarForum 已经具备较完整的跨平台客户端基础，重点特色在于桌面分栏体验、主题与语言设置、以及围绕 Flarum 内容呈现的体验优化。后续适合继续补齐自动化测试、文档、更多模块覆盖与发行流程，让项目从“能用”进一步走向“稳定可维护”。
